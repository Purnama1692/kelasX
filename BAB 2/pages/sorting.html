Apa itu Sorting?

Sorting atau pengurutan adalah proses menyusun data agar teratur menurut aturan tertentu, seperti dari yang terkecil ke terbesar.

Contoh: Insertion Sort (Interaktif)

Penjelasan Tambahan

Sorting digunakan untuk mengurutkan data sehingga lebih mudah dicari, ditampilkan, atau dianalisis. Dalam komputasi, ada berbagai algoritma pengurutan.

  • Bubble Sort: Menukar dua elemen berurutan jika dalam urutan yang salah, berulang hingga tidak ada lagi yang perlu ditukar.
  • Insertion Sort: Menyisipkan elemen ke posisi yang sesuai dalam daftar yang telah terurut.
  • Selection Sort: Memilih elemen terkecil dan menempatkannya di awal, lalu mengulang untuk sisa data.

Pemilihan algoritma tergantung pada ukuran data dan kebutuhan efisiensi.

Contoh Bubble Sort

Misal: data = [5, 3, 8, 4, 2]

  1. Pass 1: (5,3) → tukar → [3,5,8,4,2]; (5,8) → tidak; (8,4) → tukar → [3,5,4,8,2]; (8,2) → tukar → [3,5,4,2,8]
  2. Pass 2: (3,5) → tidak; (5,4) → tukar → [3,4,5,2,8]; (5,2) → tukar → [3,4,2,5,8]
  3. Pass 3: (3,4) → tidak; (4,2) → tukar → [3,2,4,5,8]
  4. Pass 4: (3,2) → tukar → [2,3,4,5,8]

Data kini terurut menaik.

Contoh Selection Sort

Misal: data = [5, 3, 8, 4, 2]

  1. Langkah 1: cari terkecil (2), tukar dengan indeks 0 → [2, 3, 8, 4, 5]
  2. Langkah 2: cari terkecil dari indeks 1→ (3) → sudah benar
  3. Langkah 3: dari indeks 2→ (4), tukar dengan 8 → [2, 3, 4, 8, 5]
  4. Langkah 4: dari indeks 3→ (5), tukar dengan 8 → [2, 3, 4, 5, 8]

Data kini sudah urut.
