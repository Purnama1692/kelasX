Praktik Mail Merge

Langkah Mail Merge

  1. Buat data di Excel (nama, alamat, dsb).
  2. Buka dokumen Word, pilih Mailings → Select Recipients → Use Existing List.
  3. Sisipkan field yang sesuai seperti <<Nama>> dan <<Alamat>>.
  4. Gunakan Preview Results untuk melihat hasil personalisasi.

Contoh: Surat undangan rapat untuk tiap ketua kelas dapat dibuat hanya dengan satu dokumen dan data siswa di Excel.
