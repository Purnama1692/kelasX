Object Linking and Embedding (OLE)

Penggunaan OLE

OLE memungkinkan objek Excel disematkan di Word atau PowerPoint dengan hubungan langsung ke sumber data.

  1. Salin tabel dari Excel.
  2. Di Word atau PowerPoint, klik kanan → Paste Special → pilih Link.
  3. Setiap perubahan data di Excel akan otomatis diperbarui di Word/PowerPoint.

Keuntungan: Tidak perlu update manual saat data sumber berubah.

Langkah-Langkah OLE

  1. Salin tabel dari Excel.
  2. Buka dokumen Word / PowerPoint.
  3. Gunakan Paste Special → pilih “Link” dan sesuaikan format.
  4. Ubah data di Excel dan lihat efek otomatis di dokumen tujuan.
